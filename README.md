## How to install the resume cli and a theme
```bash
npm install resume-cli jsonresume-theme-dev-ats
```
## How to serve the resume as HTML with a specific theme
```bash
npx resume serve --theme dev-ats
```
## To export it as pdf with a specific theme
```bash
npx resume export resume.pdf --theme dev-ats
```
